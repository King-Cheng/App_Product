智能医生助手

在线 · 响应时间 <1秒

您好！我是智能医生助手，很高兴为您提供健康咨询服务。请问您有什么不适症状或健康问题需要咨询吗？

医生，我最近总是感觉头痛，特别是早上起床的时候，已经持续一周了。

感谢您的描述。为了能够更好地了解您的情况，请问：

  • 头痛的部位在哪里？(例如：前额、太阳穴、后脑勺等)
  • 头痛的性质是怎样的？(例如：刺痛、胀痛、跳痛等)
  • 疼痛的程度如何？(1-10分，10分为最痛)
  • 最近工作或生活压力大吗？睡眠质量如何？
  • 是否有伴随其他症状，如恶心、呕吐、视力模糊等？

头痛主要在前额和太阳穴，是一种胀痛，大概5-6分的程度。最近工作压力比较大，经常熬夜到很晚，睡眠质量不太好。偶尔会有轻微的恶心感，但没有呕吐。

根据您的描述，您的头痛可能与工作压力大、睡眠不足有关，属于紧张性头痛的表现。以下是一些建议：

调整作息：尽量保持规律的作息时间，每天保证7-8小时的睡眠时间

减轻压力：适当进行放松活动，如冥想、深呼吸、瑜伽等

注意姿势：工作时保持正确的坐姿，避免长时间低头看电脑或手机

饮食调整：减少咖啡因和酒精摄入，多喝水，保持饮食均衡

如果头痛持续加重或出现其他症状，建议您尽快就医进行详细检查，排除其他潜在疾病。

那我需要吃什么药物来缓解吗？

对于轻度到中度的紧张性头痛，可以考虑以下药物缓解：

对乙酰氨基酚

成人剂量：500-1000mg，每4-6小时一次，24小时内不超过4000mg

推荐

布洛芬

成人剂量：200-400mg，每4-6小时一次，24小时内不超过1200mg

备选

请按照药品说明书或在医生指导下使用。如果用药后症状没有缓解，或者头痛频繁发作，建议您咨询医生获取更专业的建议。

药物只能缓解症状，不能解决根本问题。建议您优先调整生活方式和工作习惯。

JavaScript SDK调用示例

// 引入医疗AI SDK
const MedicalAISDK = require('medical-ai-sdk');

// 初始化SDK
const medicalAI = new MedicalAISDK({
    apiKey: 'your_api_key_here',
    secret: 'your_secret_here'
});

// 创建对话会话
async function createConversation() {
    try {
        const conversation = await medicalAI.chatbot.createConversation({
            scenario: 'general_medical_consultation', // 对话场景
            language: 'zh-CN', // 语言
            userInfo: {
                age: 30,
                gender: 'male'
            },
            systemPrompt: '你是一位专业的医疗助手，请用简洁友好的语言回答用户的问题。'
        });
        
        return conversation.sessionId;
    } catch (error) {
        console.error('创建对话失败:', error);
        throw error;
    }
}

// 发送消息并获取回复
async function sendMessage(sessionId, message) {
    try {
        const response = await medicalAI.chatbot.sendMessage({
            sessionId: sessionId,
            message: message,
            options: {
                temperature: 0.7,
                maxTokens: 1024
            }
        });
        
        return response;
    } catch (error) {
        console.error('发送消息失败:', error);
        throw error;
    }
}

// 使用示例
async function example() {
    const sessionId = await createConversation();
    const response = await sendMessage(sessionId, '我最近总是感觉头痛，该怎么办？');
    console.log('AI回复:', response.message);
}
